PROJECT TITLE:
StyleCycle

PURPOSE OF PROJECT:
Prevent textiles from going into landfills by providing consumers an easy and convenient alternative motivating them to reduce, reuse, and recycle.
We incentivize them to sell their old textiles, preventing them from entering landfills, reducing CO2 emissions, creating less demand from supply chains for production.

VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
1. Ensure the SQLite library is loaded.
2. Right click on the Main class, and run the main() function.
